# PROJECT

## 项目名称

ViroScope OS

Virus Research Operating System

## Mission

建立一套可复用的病毒进化研究系统，先从流感系统发育分析开始，后续扩展到疫苗分析、突变解释和AI辅助科研。

## Vision

用同一套平台，把原始序列数据逐步转化为可解释的演化结果和可投稿的图表输出。

## Objectives

- 标准化系统发育分析流程
- 把科研逻辑保存在文档与规范中，而不只是在代码里
- 减少重复性手工分析
- 累积可复用的谱系、突变和疫苗知识

## Supported Virus

当前聚焦：

- Influenza A

后续扩展：

- IHNV
- NDV
- SARS-CoV-2

## Target Users

- 病毒学研究者
- 研究生
- 生信协作者
- AI辅助科研流程

## Core Features

- 序列与元数据整理
- QC
- 序列比对
- 建树
- 注释
- 可视化
- 突变解释
- 报告生成

## Research Scope

第一阶段的具体范围：

- Influenza A
- H3N2
- HA基因
- ML系统发育树
- 谱系注释
- WHO疫苗株对比

## Long-Term Plan

先把它做成一个稳定的流感分析框架，再逐步扩展成支持多病毒研究的科研操作系统。
